# Luminous — Business & Digital Agency Landing Page

A modern, fully responsive landing page built for digital agencies and business consultancies. Designed to communicate growth, strategy, and technical credibility through clean visuals, structured content sections, and smooth interactions.

**Live Demo:** https://heartfelt-baklava-e67bb7.netlify.app/

## Overview

This project is a static business landing page built with vanilla HTML, CSS, and JavaScript — no frameworks, no build tools. It showcases how a fully custom, framework-free frontend can still deliver a polished, production-ready marketing site with rich sections and interactive elements.

Ideal as a starting template for agencies, startups, freelancers, or consultants who need a professional web presence without the overhead of a JS framework.

## Features

- Fully responsive layout across desktop, tablet, and mobile
- Hero section with clear value proposition and call-to-action
- About section highlighting company strengths and mission
- Services section (UI/UX Design, Web Development, Branding)
- Portfolio/Projects showcase grid
- Client testimonials section
- FAQ accordion for common client questions
- Blog/articles preview section
- Newsletter signup and contact call-to-action
- Clean, semantic HTML structure for easy customization

## Tech Stack

- HTML5 — semantic markup
- CSS3 — custom styling, responsive layout (Flexbox/Grid)
- JavaScript (Vanilla) — interactivity (accordion, navigation, etc.)
- Hosted on Netlify

## Getting Started

1. Clone the repository
   ```
   git clone https://github.com/NSniha/business-landing-page.git
   ```
2. Navigate into the project folder
   ```
   cd business-landing-page
   ```
3. Open `index.html` directly in your browser, or serve it with a local server (e.g. Live Server extension in VS Code) for the best experience.

No build step, no dependencies, no installation required.
